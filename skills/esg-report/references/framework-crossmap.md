## Table 4: 프레임워크 간 중복/보완 관계

```
ISSB S1/S2 (글로벌 기준선)
  ├── TCFD (흡수됨 → S2의 기반)
  ├── SASB (통합됨 → 산업별 지침)
  ├── CDSB (통합됨)
  └── KSSB (한국 적용)

GRI Standards (영향 중대성)
  ├── ESRS (GRI 기반 개발)
  ├── KRX 가이던스 (GRI 참조)
  └── K-ESG (GRI 매핑)

GHG Protocol (배출량 산정)
  ├── ISSB S2 (Scope 1/2/3 기준)
  ├── SBTi (감축 목표)
  ├── CDP (보고 기준)
  ├── PCAF (금융 탄소회계)
  └── 배출권거래제/목표관리제 (한국 MRV)

Double Materiality = GRI (Impact) + ISSB (Financial)
  → ESRS/CSRD가 이 구조를 채택
  → KSSB는 우선 Financial Materiality, 향후 확대 가능
```
