# Bilingual ESG Glossary — 한영 ESG 용어집

## 포맷 규칙

### Markdown 이중언어 포맷:
```markdown
## 4.1 기후변화 대응 / Climate Change Response

[한국어 본문]

> **EN** | [English translation]

**관련 프레임워크**: `GRI 305` `ISSB S2` `TCFD` `K-ESG E-1`
```

### HTML 이중언어 포맷:
```html
<section class="bilingual" id="section-4-1">
  <h2>
    <span class="lang-kr">4.1 기후변화 대응</span>
    <span class="lang-en">4.1 Climate Change Response</span>
  </h2>
  <div class="content-kr">[한국어 본문]</div>
  <div class="content-en">[English translation]</div>
  <div class="framework-tags">
    <span class="tag tag-gri">GRI 305</span>
    <span class="tag tag-issb">ISSB S2</span>
  </div>
</section>
```

### 용어 표기 규칙:
1. **프레임워크 명칭**: 영문 원문 사용, 첫 등장 시 한국어 병기
   - 예: "TCFD(기후 관련 재무정보공시 태스크포스)"
2. **약어**: 첫 등장 시 풀네임 + 약어, 이후 약어만
   - 예: "온실가스(Greenhouse Gas, GHG)" → 이후 "GHG"
3. **수치**: 한국어/영어 동일 (단위만 병기)
   - 예: "5,200,000 tCO2e"
4. **표/그래프**: 한영 헤더 병기
   - 예: "배출량 / Emissions"
